This tutorial has been replaced by the current reducer-native walkthroughs.

Use [Ring Arena](/docs/tutorials/ring-arena) for the first complete authored
game loop, or [Star Settlers](/docs/tutorials/star-settlers)
for a board-game example with typed surfaces, prompts, and trade-style
interactions.

The old version used removed APIs such as `defineAction`, `defineFlow`,
`fx.rollDie`, and `useActions`. Current games use `defineInteraction`,
collector-backed inputs, reducer-authored presentation metadata, and the
generated `WorkspaceGameShell`.
